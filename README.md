# Gated Access
Only allow defined commands to be run on your server via RCON.

## Config:

|Value|Type|Default|Description|
|--|--|--|--|
|allowedCommands|string|list|A comma-separated array of commands  to allow.|
|enabled|bool|false|Whether or not to engage the firewall.|
